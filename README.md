# @amlplugins/twilio-sync

Native `twilio` SDK re-exported under the `@amlplugins` namespace.

> Twilio Sync — real-time state synchronization across devices via Documents, Lists, Maps, Streams.

## Install

```bash
npm install @amlplugins/twilio-sync
```

## Use

```ts
import { /* named exports from twilio */ } from "@amlplugins/twilio-sync";
// or
import sdk from "@amlplugins/twilio-sync";
```

Built and published by GitHub Actions on every push to `main`.

Part of the amlplugins family — see https://github.com/amlplugins.
